# Course Scoring Application

The Course Scoring Application is designed to track events, courses, and athlete performance. It allows users to explore events, view athlete participation, and analyze scoring data across different competitions.

## Inspiration

I've been working on this project on and off for years. Its primary purpose is to serve as a training ground where I can continually refine my architecture. In fact, this project has served as a reference architecture for other applications I've launched, so it has already provided significant value. While many sites exist that track events, display leaderboards, show brackets, and publish race results, I bring a unique take to this domain. ## Technology Stack

This project was built using a modern, cloud-native technology stack, leveraging infrastructure-as-code with Bicep templates. The backend is written in **C#** and **SQL Server**, while the frontend uses **Next.js**, **shadcn/ui**, and **Tailwind CSS**. The application is hosted on **Azure**, with deployment automated via **GitHub Actions**.
