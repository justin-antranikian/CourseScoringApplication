# Course Scoring Application

The Course Scoring Application is designed to track events, courses, and athlete performance. It allows users to explore events, view athlete participation, and analyze scoring data across different competitions.

## Potential Real-World Use Cases

While this site isn't currently being used by any organization, there are many features that could be valuable. Analytics-focused companies could benefit from the comparison and performance-tracking features. Vendors could showcase training programs and goals for specific events. There are many other potential applications as well.

## Technology Stack

This project was built using a modern, cloud-native technology stack focused on scalability, maintainability, and developer productivity. The backend is written in **C#** and **SQL Server**, while the frontend uses **Next.js**, **shadcn/ui**, and **Tailwind CSS**. The application is hosted on **Azure**, with deployment automated via **GitHub Actions**.
